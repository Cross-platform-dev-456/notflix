# Welcome to Notflix Documentation

**Notflix** is a cross-platform Flutter application for discovering movies and TV shows, inspired by Netflix.

## About Notflix

Notflix is a movie and TV show discovery app that allows users to:
- Browse content by genre and category
- Search movies and TV shows
- View detailed information and trailers
- Create personalized watch lists
- Track viewing history

**Platform Support:**
- Android
- iOS
- Web
- Windows
- macOS
- Linux
